Atributos HTML




São palavras chaves que utilizamos dentro de tags de abertura para configurar aquela determinada tag.

Estes atributos carregam uma aparência semelhante, eles sempre tem uma palavra, um sinal de igual e duas áspas duplas.


<tag atributo="algumacoisa"></tag>







nome="ismael"




Existem duas classificações de atributos HTML




Atributos Públicos

São atributos que funcionam dentro de qualquer tag HTML.

Exemplo:

name=""

Este atributo serve para você colocar um nome
para qualquer tag html.



Atributos Privados


são atributos que só funcionam ou só tem sentido em determinadas tags HTML.


exemplo:


charset=""

Este atributo serve para setar o tipo de caractere utilizado naquele site, este atributo é específico da tag <meta> do HTML.


Regra: A tag <meta> só funciona dentro do espaço da tag <head> do html.

HTTP -> HyperText Transfer Protocol

É um sistema/programa/"protocolo" que funciona na internet, cujo o objetivo dele é transferir informações de um lugar para outro lugar.





Requisição e Resposta HTTP


Em comparação, requisição HTTP é como se fosse os pedidos de clientes sendos anotados pelo garçom.


As respostas é a entrega que o garçom faz para o cliente.


Poliglota -> É uma pessoa que fala em diversas línguas diferentes.


Métodos HTTP


São maneiras diferentes de se fazer uma requisição HTTP.

Servidor é um computador potente que tem como objetivo servir/entregar algo para outros computadores/sistemas/clientes.


Na verdade todo e qualquer site que existe está dentro de um computador chamado Servidor.



Existe uma curiosidade, também tem como enviar dados utilizando o método HTTP GET, desde que seja apenas string/texto/palavras.

Um detalhe interessante é que quando enviamos estas informações utilizando método GET os dados aparecem na URL, quando enviamos utilizando POST não aparecem na URL.
